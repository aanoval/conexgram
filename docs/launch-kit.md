## Launch post

I built Conexgram: a Telegram bridge for Codex CLI.

It lets you keep Codex running on your own machine and manage sessions from your phone:

- persistent Codex sessions
- Telegram `/settings` panel
- typing and progress indicators
- file upload/download helpers
- safe workspace defaults
- optional Computer Access mode
- macOS/Linux/Windows auto-start

Install:

```bash
curl -fsSL https://conexgram.com/install.sh | bash
```

Repo: https://github.com/aanoval/conexgram
