# Demo Guide

A strong launch demo should show Conexgram in under one minute.

## Suggested flow

1. Install:

```bash
pipx install conexgram
conexgram-gateway setup
conexgram-gateway install-service
```

2. Open Telegram and send:

```text
/settings
```

3. Tap a preset button, such as `Work` or `Fast`.

4. Ask Codex:

```text
Inspect this project and summarize what needs attention before release.
```

5. Show typing/progress indicators.

6. Show the final Codex response.

## Assets to capture

- Screenshot of `/settings`
- Screenshot of inline keyboard buttons
- Screenshot of a long-running progress message
- Screenshot of a final Codex answer
- Short GIF or MP4 for the README
