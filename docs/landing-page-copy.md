## Feature sections

### Persistent Codex sessions

Continue real Codex CLI sessions from Telegram without losing context between messages.

### Built for local control

Your code and credentials stay on your own machine. Conexgram bridges Telegram to your local Codex CLI.

### Safe by default

Workspace mode is the recommended default. Computer Access is optional, disabled by default, and requires explicit confirmation.

### Mobile-friendly controls

Use `/settings`, presets, typing indicators, progress messages, file upload, and `/tail` from Telegram.

### Auto-start anywhere

Install as a user service on macOS, Linux, or Windows.

## Security block

Conexgram is remote-control software. It requires a Telegram allowlist, keeps Computer Access disabled by default, and documents the security model clearly.
